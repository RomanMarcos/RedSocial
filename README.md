# MERN STACK

Red social: proyecto de curso Udemy

## Pasos para levantar el proyecto

## Clonar proyecto

- git clone git@github.com:RomanMarcos/MERN-STACK.git

## DOCKER (prerequisitos: docker & docker-compose instalados)

- cd MERN-STACK
- docker-compose up

## Blog API

- cd api
- npm install
- npm start

## Blog web app (Vite + React)

- cd blog-webapp
- npm install
- npm start
